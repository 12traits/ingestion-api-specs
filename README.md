Production: http://docs.api.12traits.com

## Deploy to GitHub Pages

```bash
bundle install
./deploy.sh
```

Custom domain is configured in https://github.com/12traits/ingestion-api-specs/settings using CNAME of `docs.api.12traits.com`.
